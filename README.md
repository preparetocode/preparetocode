## Developing locally

### Clone and install dependancies

```
git clone git@github.com:preparetocode/preparetocode.git
cd preparetocode
bundle
```

### Run

```
middleman s
```

### Deployment

Currently deployment is manual and only available to admins

```
rake deploy
```
